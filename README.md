# Jogo de Adivinhação em Python

Projeto simples desenvolvido em Python onde o jogador deve adivinhar um número aleatório gerado pelo sistema.

# Descrição

O programa gera automaticamente um número aleatório entre 1 e 100.  
O usuário deve tentar adivinhar esse número, recebendo dicas a cada tentativa.

O jogo continua até que o jogador acerte o número correto.

# Funcionalidades

- Geração de número aleatório com `random`
- Contador de tentativas
- Dicas em tempo real:
  - "Muito alto"
  - "Muito baixo"
- Tratamento de erros para entradas inválidas
- Mensagem de sucesso ao acertar

# Tecnologias utilizadas

- Python 3
- Biblioteca padrão `random`
